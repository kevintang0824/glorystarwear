Can GloryStarWear make marathon event apparel?

Yes. GloryStarWear can make marathon shirts, running singlets, finisher tees, volunteer shirts, socks, caps, towels, labels, and event packing.

Can event shirts include sponsor artwork?

Yes. Sponsor logos, charity marks, event names, race dates, sleeve marks, back prints, neck labels, and hangtags can be customized.

What should an event organizer send?

Send product list, event date, quantity, size ratio, sponsor artwork, delivery country, packaging needs, and any warehouse labeling rules.

Can marathon apparel be packed by event wave or size?

Yes. Orders can be packed by size, event wave, participant bundle, sponsor group, volunteer team, or merchandise SKU.
