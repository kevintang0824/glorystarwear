## Image Usage Rules

- Keep the final images inside `assets/images/`.
- Do not reference external generated-image folders from HTML.
- Prefer `1672 x 941` landscape images for product and hero cards.
- Use unique images for adjacent cards whenever possible.
- Update alt text when replacing an image.
- Add important new images to `sitemap.xml` for image SEO.
