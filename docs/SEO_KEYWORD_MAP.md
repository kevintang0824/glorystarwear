# GloryStarWear Search Intent Map

Last updated: 2026-07-14

## Page Ownership Rule

- A broad category page owns collection scope, buyer type, program planning, product architecture, and commercial handoff.
- A narrow product page owns garment construction, specification decisions, sample checks, fit, artwork, and product-specific quote inputs.
- Broad and narrow pages should link to each other with language that explains the different task.
- Do not reuse the same H1, section headings, direct answer, or FAQ question across both pages.
- If two pages cannot maintain different buyer tasks, consolidate them instead of publishing another keyword variation.

## Review Trigger

Review a pair when both pages gain impressions for the same query but neither develops stable clicks, or when their visible text similarity rises because a shared template has replaced product-specific information. Check title, H1, direct answer, unique section headings, internal anchor text, and the next action before changing canonical URLs.
